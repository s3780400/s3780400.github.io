Project Idea

Overview: A device manager is a control panel application in your Operating System that allows you to view and control hardware attached to your system. Device Manager highlights any piece of hardware (in this case a phone) and enables you to access the device from afar. In our application it grants the user the ability to locate, secure and remove information from their phone.
Motivation: In today’s generation almost everyone in the world owns some sort of smartphone and therefore, It is a common problem that many people across the globe end up losing their phone due to it being stolen or misplaced. According to the Consumer Reports article, 3.1 million Americans had their phone stolen (in 2013) and this does not take into account the users who did not participate in the survey and other countries across the world, thus it is extremely important to not overlook this common problem. That is why if an application was created that allows the user to locate, lock and erase sensitive information about the user, individuals won’t have to worry about the unwanted consequences that may occur if such an event were to happen.
Description: In order for the application to be in play, users must first download/install the application onto their device and connect it to their respective Google account. Once it is accomplished successfully, the user can easily track and navigate their device through Google maps even if it’s constantly on the move, thus allowing for 24/7 tracking. When the device is unable to be found, 3 options are available to the user- locking, call or reset. To perform these actions the user must visit the website and enter in their Google account allowing the website to directly connect to the user’s phone. Once this is accomplished, the user can change the PIN or password of the phone. A message is displayed showing a relatives/friends contact number to help return the users device. In addition, ringing the device will be heard in maximum volume regardless of whether it is muted or not- the phone will continue to ring until the user cancels or the battery is less than 20%. For instances, where the phone is 20% the device will ring for 15 seconds until automatically turning off and leaving a voicemail. If the phone is completely out of reach and the other 2 options have been done the user can completely reset the device back to factory settings even if the phone is off, removing all information about the user. However, because this is done the user will no longer be able to access the phone through the website and is now free to use by someone else. So although your personal information has now been entirely erased someone who who stolen it will now have access to a new smartphone and can claim it as their own, this can be seen as a drawback of this application.
Tools and Technologies: A service that grants the ability to navigate the terrain and track the location of the user's phone(i.e. Google maps). A software that allows for the development of the applcation (i.e. Android Studio, App inventor). Furthermore, softwares such as Java Development Kit, which provides package of tools for developing Java-based software.
External hardware but are not limited to: Mouse, Keyboard, Monitor, Laptop, Phone.
Internal hardware but are not limited to: CPU, Modem, Motherboard, Power Supply, RAM, Video Card
Skills Required:As the application is compatible with both iPhone and Android, appropriate coding and software will be needed to allow the functioning of both smartphones respectively, thus it is a necessity to learn multiple coding languages (i.e. JavaScript, C++, Python etc). In most cases software required for the development of the application are free, many of the resources avaliable can be accessed online, however, developing such skill sets require time,effort and dedication. Therefore, recuiting other people with the required skill may be expensive depending on the skillsets of each individual.
Outcome: If such a success were to happen, many people throughout the world would feel at ease knowing that if their phone were ever stolen or misplaced users can locate them, change security settings and if needed completely erase their personal information. Perhaps in the long-term it may de-incentivize people from stealing phones, save people money by reducing the cost of purchasing new phones and overall making the world a better place.
